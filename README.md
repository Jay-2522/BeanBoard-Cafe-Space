## Overview

**Bean Board Coffee** is a modern, fully responsive café website featuring a menu, 
workshops, story, and locations — built with a focus on clean UI and smooth 
performance.

## Installation

```bash
# Clone the repository
git clone https://github.com/Jay-2522/BeanBoard---Caf-.git

# Navigate into the project directory
cd BeanBoard---Caf-

# Install all dependencies using Bun
bun install
```

---

## Running the App

Start the local development server:

```bash
bun run dev
```

Open your browser and visit:

**[Click Here to See →](https://bean-board-cafe-space.vercel.app/)**

To create a production build:

```bash
bun run build
bun run preview
```

---

## Testing

Run unit tests with Vitest:

```bash
bun run test
```

Run end-to-end tests with Playwright:

```bash
bunx playwright test
```
